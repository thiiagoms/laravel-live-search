### Install

01 - Clone this repository:
```bash
$ git clone https://github.com/thiiagoms/laravel-live-search
```

02 - Change to this repository:
```bash
$ cd laravel-live-search
laravel-live-search $
```

03 - Execute `setup.sh`:
```bash
laravel-live-search $ chmod +x setup.sh
laravel-live-search $ ./setup.sh
```
